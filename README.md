# 🧠 Smart Data Extractor Tool

A powerful and interactive **Data Extraction Web App** built using **HTML, CSS, and JavaScript**.  
This tool allows users to input any mixed data and automatically extract useful information like emails, phone numbers, URLs, numbers, words, and more.

## ✨ Features

* 📧 Extract Emails  
* 📞 Detect Phone Numbers (India format supported)  
* 🌐 Extract URLs (web links)  
* 🔢 Extract Numbers  
* 🔤 Extract Words  
* 🔁 Detect Repeated Values  
* ✨ Show Unique Values  
* 📋 Copy-to-clipboard for each section  
* 📱 Fully Responsive Design
